# Free to Edit!
This repository contains **Python** (in **Jupyter Notebook**), **R** (including **R script** and **R Markdown**), and **SQL** (in 
**RSQLite**) projects using various data analytic techniques. It will also include projects in other programming languages in the 
future. It covers multidisciplinary topics but mainly focuses on finance-related ones. The **xlsx** files contain relevant works in 
**Excel**. Everyone who is interested in this repository is free to edit the codes.
